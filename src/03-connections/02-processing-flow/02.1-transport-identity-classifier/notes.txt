Technologies and integrations (documented, not runtime):
- Transport: Twilio webhook normalizes inbound SMS/MMS into InputRecognition types.
- Identity: In-memory Map lookup — messaging identity → Entity with type, permissions, and thread memberships.
- Classifier: Anthropic Claude API with few-shot prompts and recent thread context → TopicKey + intent classification.
- Handoff: Classified item becomes a PendingQueueItem and is enqueued via BullMQ.
