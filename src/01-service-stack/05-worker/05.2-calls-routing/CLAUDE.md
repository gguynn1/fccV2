## Step 2 — Who Does It Concern?

```
One adult ---------> their private thread
One child ---------> their private thread
The couple --------> couple thread
The whole family --> family thread
A pet -------------> responsible adult's thread
```

## Step 3 — Response or Proactive?

```
Response ----------> reply in the same
                     thread it came from
Proactive ----------> send to the narrowest
                      thread that fits
                      the audience
```

## Routing Rules

Two rules govern every message:

**Rule 1 — Responses stay in context.** If someone says something in a thread, the assistant replies in that same thread. Everyone present sees it.

**Rule 2 — Proactive messages route to the narrowest appropriate thread.** When the assistant initiates, it sends to the smallest thread that includes only the people who need the information. A chore reminder for PARTICIPANT 3 goes to their private thread. A bill alert goes to the couple thread. A health follow-up goes to the individual's private thread.

When both rules apply to the same event — someone says something in a group thread that will later need a proactive follow-up — Rule 1 handles the immediate reply and Rule 2 handles any future outbound.
